Frontend - Gestión de Eventos

- Descripción

Aplicación Angular que consume la API de eventos.
Permite visualizar y crear eventos mediante Angular Material.

- Tecnologías

Angular (Standalone Components)

Angular Material

TypeScript

- Instalación

Instalar dependencias

npm install

Ejecutar aplicación

npm run dev

o

ng serve

Corre en:

http://localhost:4200
